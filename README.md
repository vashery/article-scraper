# article-scraper

This article scraper app is a simple app that lets you see articles and comment on them

## Requirements

nodejs
mongodb

## Installation

Use npm to install the software.

```bash
npm install
```

## Usage

Modify port if needed and add database credentials

```bash
node server.js
```
